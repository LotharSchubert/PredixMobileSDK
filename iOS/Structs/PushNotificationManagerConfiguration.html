PushNotificationManagerConfiguration

public struct PushNotificationManagerConfiguration

Configuration for Push Notification Authentication

  • init()

    Initialization:

    PushNotificationManagerConfiguration will attempt to read oAuth client id and client secret values from the Info.plist using the keys client_id and client_secret

    Declaration

    Swift

    public init()
  • pushUAATokenUrl

    The issuerId UAA server URL to use for online authentication (ending in /oauth/token in general). For example:

    push_uaa_token_url is set to : https://mybackend-uaa.predix.io/oauth/token in Info.plist

    Declaration

    Swift

    public var pushUAATokenUrl: URL?
  • pushClientId

    Client ID value to use for online authentication. A client ID is a public identifier for the authentication server to identify the App

    default: Value loaded from push_uaa_client_id key in Info.plist

    See also

    https://www.oauth.com/oauth2-servers/client-registration/client-id-secret/

    Declaration

    Swift

    public var pushClientId: String?
  • pushClientSecret

    Client secret value to use for online authentication. A client secret should be known only to the application and the authentication server

    default: Value loaded from push_uaa_client_secret key in Info.plist

    See also

    https://www.oauth.com/oauth2-servers/client-registration/client-id-secret/

    Declaration

    Swift

    public var pushClientSecret: String?
  • pushServiceUrl

    URL endpoint where Push Notification Service instance is deployed in Predix For example:

    push_service_url is set to : https://mybackend-push.predix.io/ in Info.plist default: Value loaded from push_service_url key in Info.plist

    Declaration

    Swift

    public var pushServiceUrl: URL?
  • pushAppKey

    App Key that was generated as part of registration of application (bundleId) in Push Notification Service instance

    default: Value loaded from "push_app_key" key in Info.plist
