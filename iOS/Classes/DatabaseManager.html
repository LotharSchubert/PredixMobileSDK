DatabaseManager

public class DatabaseManager: NSObject, DatabaseManagerProtocol

Database interaction class. Controls setup of replication, access to documents, manipulation of attachments, etc.

Public DatabaseManagerProtocol methods
  • setupReplication(forDatabase:withUrl:onComplete:firstSyncComplete:)

    Sets up initial database replication.

    Declaration

    Swift

    public static func setupReplication(forDatabase database: String, withUrl url: URL, onComplete: @escaping taskCompleteBlock, firstSyncComplete syncComplete: (()->())?)

    Parameters

    forDatabase

    Name of database to start replicating. Currently must match database name in PredixMobilityConfiguration.defaultDatabaseName or the token “~”
    withUrl

    URL of replication gateway
    onComplete

    Closure, incdicating success or failure of setting up replication. Success does NOT mean replication has completed successfully, or that the URL provided was valid or reachable, just that there were no internal errors in initializing the replication process.
    firstSyncComplete

    Optional closure that, if provided, will be called when the initial replication process has completed successfully.

    Adds or removes given channel names to pull (server to client) replication

    onComplete closure is defined as:

    (success: Bool, error: Error?)->()

    Success will be false if both channelsToAdd and channelsToRemove is nil or an empty array.

    Declaration

    Swift

    public static func updatePullReplicationChannels(channelsToAdd: [String]?, channelsToRemove: [String]?, onComplete: @escaping taskCompleteBlock)

    Parameters

    channelsToAdd

    String array of channels to add. Pull replication will be restricted to only these channels
    channelsToRemove

    String array of channels to remove. Removing all channels will allow replication of all data to which the user has access.
    onComplete

    Closure called when assignment of channels is complete.
